# react-tailwind-lib

## React TailwindCSS library

Component sizes:
 - `sm`
 - `md`
 - `lg`



## Buttons
Features:
* handle states:
  - [ ] disabled
  - [ ] loading
  - [ ] active
  - [ ] hover
  - [ ] focus (with outline)

* 3 variants:
  - [ ] text
  - [ ] outlined (secondary)
  - [ ] contained (primary)

* 3 roundness levels:
  - [ ] default (slightly rounded)
  - [ ] rounded (button pill)
  - [ ] circle (fab)
