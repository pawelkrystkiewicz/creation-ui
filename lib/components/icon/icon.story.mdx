# Icon

Gemini UI uses icons from [material design](https://fonts.google.com/icons?icon.set=Material+Icons)

<br />
To use icon just pass in the `icon` prop the name of the icon from the material design:

<Canvas withSource="open">
  <Story
    name="Icon"
    args={{
      ...defaultProps,
    }}>
    {Template.bind({})}
  </Story>
</Canvas>

<br />

## Changeing icons appearance

To change icon size, stroke, color or background-color use tailwind utility classes. For example:

<Canvas>
  <Story
    name="Font size"
    args={{
      ...defaultProps,
      className: 'text-2xl',
    }}>
    {Template.bind({})}
  </Story>
  <Story
    name="Color"
    args={{
      ...defaultProps,
      className: 'text-green-500',
    }}>
    {Template.bind({})}
  </Story>
  <Story
    name="Background"
    args={{
      ...defaultProps,
      className: 'bg-primary-500 p-1 text-white rounded',
    }}>
    {Template.bind({})}
  </Story>
</Canvas>
